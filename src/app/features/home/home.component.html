Safety instructions

Check before signing

Always review the active network, wallet address, contract address, calldata preview, and wallet confirmation before submitting transactions.

Whenever your wallet supports it, add or use named contracts for the SETHX protocol addresses. Named contracts make it easier to recognize the intended contract before you approve or sign an action.
