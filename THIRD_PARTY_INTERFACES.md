# Third-Party Interfaces, Forks, and Hosted Deployments

SETHX Client is released as local client software and source code.

SETHX Community is not currently operating a public hosted trading website for the client.

## Local client first

The initial distribution model is local-first:

- developers can run the source code locally;
- testers can use a local browser release;
- users interact through their own wallet software;
- contract addresses and network configuration must be verified before use.

This model supports transparency, review, and community testing while avoiding premature operation of a public hosted trading interface.

## Public hosted websites

A public hosted trading interface can create additional responsibilities, including:

- legal and regulatory compliance;
- jurisdiction and regional access decisions;
- user disclosures and risk notices;
- privacy and analytics practices;
- security monitoring;
- infrastructure operations;
- user support;
- marketing and advertising compliance.

For that reason, SETHX Community does not currently present the client as a public hosted trading website.

The public website at `sethx.com` is intended for information, documentation, community updates, and contributor onboarding. It is not the hosted trading client.

## Third-party use

The SETHX Client source code may be used, forked, modified, or extended according to the applicable project license.

However, any person, team, company, organization, or community that creates, modifies, deploys, hosts, promotes, or operates a third-party website, fork, hosted interface, modified client, public deployment, or related service is responsible for their own:

- legal and regulatory compliance;
- jurisdiction and regional access decisions;
- user disclosures and risk notices;
- security practices;
- privacy and analytics practices;
- infrastructure and hosting choices;
- contract and address configuration;
- wallet connection behavior;
- transaction construction and display;
- user support;
- marketing and advertising compliance;
- tax, reporting, or business obligations where applicable.

SETHX Community is not responsible for third-party websites, forks, hosted interfaces, modified builds, unofficial deployments, external services, or independent public deployments.

## No endorsement by default

A third-party website, fork, hosted interface, modified client, or deployment is not endorsed by SETHX Community unless there is an explicit written statement from an authorized SETHX Community maintainer or governance-approved process.

Use of the SETHX name, logo, repository links, contract addresses, source code, or protocol-compatible contracts does not automatically mean that a third-party interface is official, safe, reviewed, endorsed, or maintained by SETHX Community.

## User caution

Users should be careful when using any hosted interface or modified build.

Before connecting a wallet or signing a transaction, users should verify:

- the website or software source;
- the repository and release information;
- the network;
- the contract addresses;
- the transaction details shown in the wallet;
- whether the interface is official, community-recognized, or third-party;
- whether the operator is subject to local legal requirements.

Users should never enter seed phrases or private keys into any website, app, or client.

## Community-created interfaces

Community members may propose additional interfaces, hosted deployments, or distribution models through the community process.

Before any interface is described as official or community-recognized, it should be reviewed for:

- security;
- contract/address accuracy;
- transaction clarity;
- privacy and analytics practices;
- hosting and operational responsibility;
- legal and regional considerations;
- support expectations;
- branding and communication accuracy.

## Governance and recognition

In the future, SETHX Community may define a formal process for recognizing or listing community interfaces.

Until such a process exists, users should treat external websites and deployments as independent third-party services.
